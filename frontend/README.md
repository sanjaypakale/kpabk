React application (Vite + React 19) for the KPABK Connect project.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs the app at [http://localhost:5173](http://localhost:5173). The dev server proxies `/api`, `/v3`, and `/swagger-ui.html` to the backend at `http://localhost:8080` (start the Spring Boot app for full API access).

## Build

```bash
npm run build
```

Output is in `dist/`. Use `npm run preview` to preview the production build locally.
